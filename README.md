## What is devTools?

devTools is a browser-native utility suite for developers, students, freelancers, and technical creators. It includes tools for formatting JSON, encoding and decoding Base64, inspecting JWTs, generating hashes, parsing URLs, creating QR codes, compressing images, generating invoices, renaming files, converting units, and cleaning text.

The goal is simple: fast, private, polished utilities that work directly in the browser without forcing users into a cloud workspace.

## Tools Included

- JSON Studio
- Base64 Studio
- Hash Workbench
- JWT Studio
- URL Toolkit
- QR Studio
- Image Compressor
- Invoice Generator
- File Renamer
- Unit Converter
- Text Cleaner

## Why local-first?

devTools is designed around browser-native utility work. The tools are built to be fast, private, and practical, with local session continuity, exportable runs, and no unnecessary login requirement for basic utility use.
